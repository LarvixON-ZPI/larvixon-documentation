## List of Symbols, Designations, and Acronyms

| Acronym   | Definition                                                                                                                                                                    |
| :-------- | :---------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| **AI**    | _Artificial Intelligence_.                                                                                                                                                    |
| **API**   | _Application Programming Interface_ – a set of rules for communication between computer systems.                                                                              |
| **BLoC**  | _Business Logic Component_ – a design pattern separating business logic from the presentation layer, popular in the Flutter ecosystem.                                        |
| **CI/CD** | _Continuous Integration / Continuous Delivery_ – software engineering practices involving frequent code integration, automated application deployment, and automated testing. |
| **CLI**   | _Command Line Interface_.                                                                                                                                                     |
| **CNN**   | _Convolutional Neural Network_ – a type of deep neural network specialized in image processing.                                                                               |
| **DTO**   | _Data Transfer Object_ – a simple object used to transfer data between processes or application layers.                                                                       |
| **FHIR**  | _Fast Healthcare Interoperability Resources_ – a standard for exchanging healthcare data created by the HL7 organization.                                                     |
| **GPU**   | _Graphics Processing Unit_ – utilized in the project to accelerate ML model calculations.                                                                                     |
| **HL7**   | _Health Level Seven_ – a set of international standards for the exchange of medical information.                                                                              |
| **HTTP**  | _Hypertext Transfer Protocol_ – a protocol for transmitting hypertext documents, the foundation of data communication on the WWW.                                             |
| **JWT**   | _JSON Web Token_ – a standard for securely transmitting information between parties as a JSON object, used for authorization.                                                 |
| **LSTM**  | _Long Short-Term Memory_ – a type of recurrent neural network (RNN) capable of learning long-term dependencies in sequential data.                                            |
| **ML**    | _Machine Learning_.                                                                                                                                                           |
| **ORM**   | _Object-Relational Mapping_ – a technique for mapping objects in program code to a relational database.                                                                       |
