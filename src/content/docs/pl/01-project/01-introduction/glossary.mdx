## Wykaz symboli, oznaczeń i akronimów

| Akronim   | Definicja                                                                                                                                                                                          |
| :-------- | :------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| **AI**    | _Artificial Intelligence_ – sztuczna inteligencja.                                                                                                                                                 |
| **API**   | _Application Programming Interface_ – interfejs programowania aplikacji, zestaw reguł komunikacji między systemami komputerowymi.                                                                  |
| **BLoC**  | _Business Logic Component_ – wzorzec projektowy oddzielający logikę biznesową od warstwy prezentacji, popularny w ekosystemie Flutter.                                                             |
| **CI/CD** | _Continuous Integration / Continuous Delivery_ – praktyki inżynierii oprogramowania polegające na częstym integrowaniu zmian w kodzie, automatycznym wdrażaniu aplikacji i automatycznych testach. |
| **CLI**   | _Command Line Interface_ – interfejs wiersza poleceń.                                                                                                                                              |
| **CNN**   | _Convolutional Neural Network_ – konwolucyjna sieć neuronowa, typ sieci głębokiej specjalizujący się w przetwarzaniu obrazów.                                                                      |
| **DTO**   | _Data Transfer Object_ – prosty obiekt służący do przesyłania danych między procesami lub warstwami aplikacji.                                                                                     |
| **FHIR**  | _Fast Healthcare Interoperability Resources_ – standard wymiany danych medycznych stworzony przez organizację HL7.                                                                                 |
| **GPU**   | _Graphics Processing Unit_ – procesor graficzny, wykorzystywany w projekcie do przyspieszania obliczeń modelu ML.                                                                                  |
| **HL7**   | _Health Level Seven_ – zbiór międzynarodowych standardów wymiany informacji medycznych.                                                                                                            |
| **HTTP**  | _Hypertext Transfer Protocol_ – protokół przesyłania dokumentów hipertekstowych, podstawa komunikacji w WWW.                                                                                       |
| **JWT**   | _JSON Web Token_ – standard bezpiecznego przesyłania informacji między stronami jako obiekt JSON, używany do autoryzacji.                                                                          |
| **LSTM**  | _Long Short-Term Memory_ – typ rekurencyjnej sieci neuronowej (RNN) zdolny do uczenia się długoterminowych zależności w danych sekwencyjnych.                                                      |
| **ML**    | _Machine Learning_ – uczenie maszynowe.                                                                                                                                                            |
| **ORM**   | _Object-Relational Mapping_ – technika mapowania obiektów w kodzie programu na relacyjną bazę danych.                                                                                              |

## Słownik pojęć

Poniższa tabela zawiera definicje kluczowych pojęć wykorzystywanych w dokumentacji.

| Termin              | Definicja                                                                                                                                               |
| :------------------ | :------------------------------------------------------------------------------------------------------------------------------------------------------ |
| **Drzewo widgetów** | Hierarchiczna struktura organizująca widgety w relacje rodzic-dziecko, odwzorowująca budowę interfejsu użytkownika w aplikacji.                         |
| **Ksenobiotyk**     | Substancja chemiczna obecna w organizmie, która nie jest w nim naturalnie wytwarzana ani nie występuje w nim w normalnych warunkach (np. lek, toksyna). |
| **Widget**          | Podstawowy element budulcowy interfejsu w technologii Flutter.                                                                                          |
